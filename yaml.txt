YAML - YAML Ain't Markup Language.
Язык сериализации данных. Поддерживается большим числом языков программирования и программных продуктов.

СИНТАКСИС:
---------- 
Структура задается отступами (пробелы, не TAB!)
#  Односторчный комментарий
--- начало документа
Перваый блок задает рабочее количество отступов.
Список:
 - элемент списка 1
 - элементы списка 2
 - вложенный список:
  - элемент вложенного списка 1
  - элемент вложенного списка 2
ключ: значение            # тип данных строка
ключ: "первая команда | вторая команда"
ключ: >
 Длинная строка
 ее продолжение
 ее конец.
yes              # равносилен True
Словарь:
 элемент словаря 1: значение
 элемент словаря 2: значение
... конец документа
