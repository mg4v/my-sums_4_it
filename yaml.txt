YAML - YAML Ain't Markup Language.
Язык разметки, основанный на языке программирования Python. Поддерживается большим числом языков программирования и программных продуктов.

СИНТАКСИС:
---------- 
Структура задается отступами (пробелы, не TAB!), каждый уровень вложенности начинается с новой строки и увеличивает количество пробельных символов на одну табуляцию.

Многострочные строки: Для создания многострочных строк в YAML используется обратный слеш "\". Это позволяет переносить строку на новую строку без потери информации.

Пустые значения: Пустые значения в YAML обозначаются одним пробелом после ключа.

Даты и время: YAML поддерживает даты и время в формате ISO 8601.

Загрузка файлов: YAML позволяет загружать файлы, используя относительные или абсолютные пути.

Загрузка ресурсов: YAML позволяет загружать ресурсы, используя URL-адреса.

#  Односторчный комментарий

Перваый блок задает рабочее количество отступов.

  - элемент вложенного списка 1

ключ: значение            # тип данных строка
ключ: "первая команда | вторая команда"
ключ: >
 Длинная строка

Словарь:

... конец документа
